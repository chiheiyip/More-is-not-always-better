# Raw Data

Do not commit identifiable participant data or proprietary raw exports here.

Expected raw sources:

- Questionnaire wide Excel or CSV from the survey workflow.
- Eye-tracking scene CSV files and AOI JSON files.
- EEG scene-level export from the MATLAB/EEGLAB workflow.

Use `configs/paths.local.json` to point to local real data.

Raw data are ignored by git except for this README.
