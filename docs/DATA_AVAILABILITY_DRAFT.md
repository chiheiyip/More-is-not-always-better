# Data Availability Draft

This is a Nature-style working draft. Replace every `AUTHOR_INPUT_NEEDED` field before manuscript submission.

## Ready-To-Paste Draft

Processed analysis tables, figure source data, synthetic test fixtures, and code required to reproduce the reported analyses will be deposited in a DOI-backed public repository before submission. Repository name and identifier are `AUTHOR_INPUT_NEEDED`.

De-identified processed data will include the canonical multimodal analysis table, model result tables, robustness diagnostics, reviewer-response evidence tables, and figure source data generated by this repository.

Raw questionnaire exports, raw eye-tracking scene files, AOI definitions, and raw or intermediate EEG exports may contain identifiable or consent-restricted participant information. If these files cannot be openly shared, access conditions, request review route, responsible contact, ethics constraints, and any representative public metadata are `AUTHOR_INPUT_NEEDED`.

## Repository Actions

| Dataset | Access route | Required author action |
|---|---|---|
| Raw questionnaire export | Restricted or de-identified repository | Confirm consent, de-identification, repository, and identifier. |
| Raw eye-tracking exports and AOI definitions | Restricted or de-identified repository | Confirm whether gaze files can be shared publicly after de-identification. |
| EEG scene-level export | De-identified repository or controlled access | Confirm whether processed scene-level EEG metrics are non-identifiable. |
| Analysis-ready master table | Public repository | Deposit reproducible table and cite repository record. |
| Figure and table source data | Public repository | Deposit source CSV files for each figure/table. |
